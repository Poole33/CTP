### CTP 期货行情数据获取
1. 通过上期所的提供的接口，订阅期货信息，进行数据获取，五分钟获取一次
2. 通过C++实现

1. 需要在[SimNow](http://www.simnow.com.cn/)仿真交易平台，注册测试账号

2. 项目启动可能会提示缺少一些.dll文件.可能缺少的文件，对应32位pc和64位pc，放在了32和64两个文件夹中
3. config.ini位配置文件，对应的测试账号密码和数据库账号密码填入此文件
